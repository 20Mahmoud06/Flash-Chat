## 🚀 Overview

Flash Chat is a modern real-time chat application built entirely with Flutter.
It delivers smooth messaging, voice & video calls, smart contact detection, and secure authentication — all powered by Firebase, Agora, and Cloudinary.

Designed with a custom UI/UX from scratch to provide a clean and minimal experience ⚡

---

## ✨ Features

### 🔐 Secure Authentication
- Email & password login/signup
- Firebase Authentication
- Secure password reset via email

### ⚡ Real-Time Messaging
- Instant message updates using Cloud Firestore
- Reliable and scalable architecture

### 💬 Private & Group Chats
- One-to-one messaging
- Multi-participant group conversations

### 📞 Smart Contacts Sync
- Detects which saved contacts are using Flash Chat

### 🔔 Push Notifications
- Powered by Firebase Cloud Messaging (FCM)

### 🎥📞 Audio & Video Calls
- High-quality voice calls
- Smooth video calls
- Powered by Agora SDK

### 🎙️📷 Media & Voice Notes
- Share images, videos, and voice notes
- Uploaded securely via Cloudinary

### 😀 Emoji Avatars
- Unique emoji-based identity for each user
- Fun and minimal design

### 🎨 Custom UI/UX
- Designed from scratch
- Clean, modern, lightweight interface

# 📦 Installation

Clone the repository:

```bash
git clone https://github.com/20Mahmoud06/Flash-Chat.git
```

Navigate to the project directory:

```bash
cd Flash-Chat
```

Install dependencies:

```bash
flutter pub get
```

Run the application:

```bash
flutter run
```

---

# 🔧 Firebase Setup

Add the required Firebase configuration files:

```text
android/app/google-services.json
ios/Runner/GoogleService-Info.plist
```

Make sure Firebase Authentication, Cloud Firestore, and Firebase Cloud Messaging are properly configured in your Firebase project.

---

# 🔐 Security Notice

Never upload sensitive configuration files or API keys.

The following files should remain private:

```text
agora_config.dart
cloudinary_config.dart
service_account.json
```

Keep all credentials and API keys secure.
